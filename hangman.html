The Game of Hangman

HTML, CSS, and JavaScript

This is a pretty straightforward little game. I set it up so that each guess word is chosen at random from a list I created myself, all of the test cases checking whether the letter is within the chosen word, adding a new part of the body anytime the letter is not within the word, and changing the letter to represent whether it has been played or not.
