The Car Gallery

JavaScript Frontend and Rails API Backend

I loved making this project. I learned a lot about fetching data from the backend and displaying it on the frontend.

In this app you can add a new Car, Truck, or Recreational Vehicle's Year, Make, Model, and an Image url. You also have the ability to add Upvotes, Downvotes, and delete a car. There is a default image added on to each vehicle upon creation, provided you don't add a url yourself.

You can also choose to view all of the Cars, Trucks, or Recreational Vehicles via a drop down, with a button close by to view all vehicles on the same page.
